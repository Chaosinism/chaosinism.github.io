创作工具考察

  2020-09-15   长期更新   资源汇总  评论 字数统计: 5021字 阅读时长: 11分
  • 这个页面是什么？本页面列举笔者体验过并推荐的创作工具，主要收录开源或是免费的软件。虽然之前就写过一个版本，还是希望试图进行更详细的介绍和更有条理的分类，因此对格式和内容都进行了修改。
  • 为什么关心免费？主要原因就是笔者没有钱。另外现在订阅式收费在创作工具中大行其道，实在不太喜欢这种趋势，即使价格不高也不想作茧自缚。注意列举出的部分工具存在付费后才能激活的功能，对此，本页面收录的标准是其关键功能（如保存和导出）和作品的授权在免费条件下不受限制，对于较为复杂的情况，表格中以“?”标注并附有解释。
  • 为什么关心开源？对于创作工具，个人认为只掌握操作方法并不足以称为学会技术，而开源工具方便使用者了解每个功能背后的算法，甚至允许使用者做出自己的扩展与创新。此外，笔者对开源精神与其社区文化较为认同，感觉它与同人创作的精神也有很多相通之处。虽说如此，笔者还不算开源的狂热爱好者，如果开源工具的功能明显落后于同类闭源软件，多半还是会倾向使用后者。
  • 评价的标准？基本上是根据笔者的个人使用体验，其它创作者可能会给出完全不同的评价。其中标为粗体的工具是笔者自身较常用的，在本站也可能找到其它关于这些工具的文章。

3D

名称 类别 链接 免费 开源 个人评测
Blender 建模\动画 官网 ○ ○ 精简而又全面的3D创作软件，可以独立完成从建模、动画到渲染的全过程，并且还有一定的视频剪辑、特效及绘画功能。可定制程度很高，除开源外，还可利用脚本、节点、表达式等方式灵活达到各种目的。与本体风格类似，第三方插件的思路也非常丰富。
MikuMikuDance 动画 官网 ○ x 如果进行同人创作，MMD的模型与动作资源无疑是最丰富的。虽然本体功能简单，但可以利用MME增强渲染效果或制作特效。MMD自带对Kinect的支持以进行动作捕捉，但实际上对系统版本与硬件型号等有诸多限制。
MikuMikuMoving 动画 官网 ○ x 使用MMD模型制作动画的另一选择。与MMD相比，它不兼容大多数MME，但有更强的K帧功能，包括手部IK、手指、口型等。可以仅用它制作动作，然后导入MMD或Blender中渲染。
PmxEditor 建模 官网 ○ x 专为MMD模型——PMD与PMX格式准备的编辑器。虽然功能不及专门的建模软件，但如果专注于MMD，这个工具是不可缺少的。
OpenMMD 动画 源码 ○ ○ 建立在OpenPose的基础上，利用深度学习对视频或图片进行姿势与深度识别，并生成MMD动作文件。
ALP2PMX 建模-人物 ? x 将TEATIME出品的游戏中定制的人物提取为MMD模型。虽然这个工具免费，但游戏本身不是。以现有的例子来判断，似乎非商业用途的MMD模型不怎么会被追究的样子。
MakeHuman 建模-人物 官网 ○ ○ 生成写实风格人物模型，体型与五官均可调节，此外也有用户自行贡献的服装与发型等。与Blender配合很好。
FaceGen 建模-人物 官网 ? x 由照片生成人头模型的工具，这类工具有不少，这一款相对来说较为物美价廉。如果使用免费版，模型额头会被添加水印，但是不用费太多力气就可以将它去掉。
VRoid Studio 建模-人物 官网 ○ x 动漫风格的捏人工具，制作头发比较方便，但完成后有时需要再用别的软件优化。由于注重VR领域，它主要使用的格式是VRM，但也可转换并用于其它3D软件。
Meshroom 建模-物件 官网 ○ ○ 根据多个角度拍摄的物体照片自动生成该物体的3D模型。
Sweet Home 3D 建模-场景 官网 ○ ○ 室内设计软件，定制并生成3D房间的低模。
AwesomeBump 贴图 源码 ○ ○ 处理凹凸、法线、AO、反射等多种3D模型所需的贴图。
JSplacement 贴图 官网 ○ x 生成由随机的块状、线状图案构成的贴图，对于未来风格的场景很合适。
DeepNormals 贴图 源码 ○ ○ 利用深度学习，自动由二维绘画估计出法线贴图，可以用于辅助上色或是将绘画立体化。
MMD Tools Blender插件 源码 ○ ○ 在Blender中导入或导出MMD模型与动作，并可使用模型自带的基于碰撞体积的物理模拟，不过物理这方面并不完美，有时会与MMD中的效果有区别甚至出错。
Animation Nodes Blender插件 源码 ○ ○ 虽然Blender目前致力于将粒子系统节点化，但尚未完成，这个插件则以类似的方式极大拓展了Blender动画的灵活度，尤其适合用于MG动画制作。
VSE Transform Tools Blender插件 源码 ○ ○ 目前Blender自带的视频编辑器使用起来并不方便，而这个插件简化了很多操作，尤其是可以很快速地完成镜头跟踪。

视频

名称 类别 链接 免费 开源 个人评测
Aviutl 剪辑/特效 官网 ○ ? 日式MAD与PV作者常用的视频编辑软件，虽然界面看起来很简陋，功能其实很强大。它的本体没有开源，但有众多第三方的开源插件与脚本，来增强其视频处理能力。
DaVinci Resolve 剪辑/特效 官网 ○ x 非线性视频编辑软件，以调色功能闻名，最近版本中又将基于节点的特效工具也免费化。
EbSynth 特效 官网 ○ x 自动视频风格迁移，手动处理完成几个关键帧后（描画或使用图像滤镜），该工具即可将整段视频风格化。
PhotoAnim 骨骼动画 官网 ○ x 为静态图片制作骨骼动画，或简单地将其3D化。

绘画

名称 类别 链接 免费 开源 个人评测
MediBang Paint 位图绘制 官网 ○ x 绘画软件，除桌面版外也有iOS和安卓版，并且提供云储存以便随时在不同的平台之间切换。除常规的笔刷、网点等素材外，软件还提供大量日文与英文字体，并均可免费商用。
Krita 位图绘制 官网 ○ ○ 开源绘画软件，提供十余种不同的笔刷引擎和非破坏式的滤镜。说实话个人用起来不是很习惯，特别是使用不带压感功能的触摸屏时，不知道怎样才能生成较自然或多变的笔触。兼容GIMP的插件。
Inkscape 矢量图绘制 官网 ○ ○ 矢量图的绘制软件，顺带还能用来编辑PDF文件。
Lunacy 矢量图绘制 官网 ○ x 适用于平面设计，成品可输出为矢量格式以及CSS代码。软件也可以方便地利用到ICONS8网站的大量素材。
GIMP 图像处理 官网 ○ ○ 对标Photoshop的图像处理软件，前者日常的应用大多数也都能用GIMP完成，然而它却使用另一套快捷键，对入门造成一定难度。
FotoSketcher 图像处理 官网 ○ x 风格化工具，适合将照片处理为铅笔画\油画\水彩画等手绘风格。虽然下面提到的G’MIC也有相同功能，感觉这个工具的效果好一些。
Style2paints 线稿上色 源码 ○ ○ 国人的深度学习自动上色项目。现在有一些绘画网站与软件提供了深度学习上色功能，看起来大多数的原理类似PaintsChainer，而这个项目的上色效果通常要比它们更好。
G’MIC GIMP插件 官网 ○ ○ 提供大量图片滤镜、特效，除了GIMP，也能作为OpenFX格式的视频插件或者独立运行。其中很多算法运行速度较慢，用于处理视频时尤其需要注意。
Resynthesizer GIMP插件 官网 ○ ○ GIMP没有像Photoshop那样方便的智能填充\修补功能，但这个插件可以做到。作为顺带的功能，它也可以实现风格迁移，虽然效果远不如基于深度学习的方法。

音频

名称 类别 链接 免费 开源 个人评测
UTAU 歌声合成 官网 ○ x 歌声合成工具，可定制程度很高，不仅音源与插件，连声音合成引擎都存在用户自制的版本可供选用。
Audacity 音频分析/处理 官网 ○ ○ 开源音频处理软件，擅长处理波形文件，音频的拉伸、变调、降噪、频谱分析等均可以完成，功能类似Audition。支持包含VST的多种插件格式。
Spleeter 音频处理 源码 ○ ○ 基于深度学习，将音频的人声与乐器分离，输出成单独的文件。一键即可完成操作，且效果经常要比传统方法更好。
VocalShifter 音频处理 官网 ○ x 擅长处理人声，能够识别出歌声的音调，画出频谱图，同时用户可通过画曲线的方式来自由地升高、降低或平滑音调。
PureData 音频/MIDI处理 官网 ○ ○ 开源的数字信号处理编程工具，能以流程图的形式自制合成器、采样器、效果器等大多数音频应用，不需要写代码。同时网上也可找到大量爱好者自制的程序。
PurrData 音频/MIDI处理 官网 ○ ○ 上面PureData的一个分支，整合了许多第三方插件，更重要的是改善了图形界面，开发与使用都变得方便了许多。
VCV Rack 音频处理 官网 ○ ○ 模拟了现实中的Eurorack模块化合成器，有数以千计的音频处理模块可供组合。除独立使用外，也有人制作了非官方的VST插件版本。
MixMeister BPM Analyzer 音频分析 官网 ? x 测乐曲的BPM似乎没有很完美的方法，这可能是个较好用的工具。它现在虽然便宜，但也已经不是免费。不过网上还可以找到它的早期免费版本。
Impro-Visor 乐谱分析 官网 ○ x 给定和弦后，自动生成爵士乐伴奏，既可用于乐理学习，也可辅助音乐制作。它支持多种生成规则、风格与方式，甚至不局限于爵士乐。
Cakewalk DAW 官网 ○ x 古老的DAW，近年被收购后变为了免费。它的基础功能全面，也正在加入现代化的新功能，并可用脚本扩展。软件自带不少采样与乐器，质量不错。
LMMS DAW 官网 ○ ○ 开源的DAW，操作简便，与FL Studio较为接近。它缺少很多重要的功能，比如无法处理波形、无法录音、不支持VST3等。它的开发进度也不是特别快，不知未来可以抱多大期待。
Vmpk MIDI输入/输出 官网 ○ ○ 虚拟键盘，用键盘、鼠标或触屏操作。可以直接发声，也可以产生MIDI信号输入到DAW。
LoopBe1 MIDI输入/输出 官网 ○ x 将软件之间的MIDI输入、输出互相连接，在没有MIDI硬件的情况下也能完成。可以配合上面的Vmpk或Impro-Visor，也能用在同一DAW的多个轨道之间。
DSK系列 VST乐器-采样/民族 官网 ○ x 提供十分丰富的乐器采样，包括较为少见的类型，如多种亚洲民族乐器的音色。不过音色质量参差不齐。
Kong系列 VST乐器-采样/民族 官网 ? x 为数不多的中国乐器系列，其中古筝、笛子、二胡提供免费版本，每个乐器又包含多个音色与技巧。
BBC Symphony Orchestra VST乐器-采样/管弦 官网 ? x 交响乐插件。它的入门版本目前可以通过填写调查问卷来免费获得，包含35种乐器和47种技巧，由著名的BBC交响乐团录制。在插件图形界面中可以直观地看到每个乐器的空间位置。
SpitFire LABS VST乐器-采样 官网 ○ x 这个系列既有传统乐器，也有合成器音色与实验性音色，均为专业音乐家录制而成，有很好的音质。
Virtual Playing Orchestra SFZ乐器-采样/管弦 官网 ○ x 本项目收集了多个遵循CC协议的交响乐采样，拼合成了一个更全面的音色库。不过正因如此，各乐器的空间位置感并不同，有时需要用混响等方法手动调整。
XenFont VST乐器-采样/SF2 介绍 ○ x SF2格式的采样器，同时又综合了减法合成器的一些功能，另外它支持改变调性（微分音）。一个缺点是仅有Win32位的版本。
Shortcircuit 2 VST乐器-采样 介绍 ○ x WAV文件的采样器，使用方便并且功能全面，不过已经停止开发，目前仅有32位版本，文档也无法再访问了。
Surge VST乐器-合成 官网 ○ ○ 老牌的减法合成器，现在变为开源得到更频繁的更新与维护。它自带丰富的波表并支持自定义，此外可以通过LFO对音色进行非常灵活的调制。
Synth1 VST乐器-合成/复古 官网 ○ x 非常古老的合成器，不过至今受到很多人的欢迎，可以在网上找到数以万计的预置音色，常被用来制作Synthwave等带有复古特点的音乐。
Vital VST乐器-合成/波表 官网 ○ x 新发布的波表合成器，具有灵活的参数控制和直观的可视化界面，支持微分音，另有文本转换波表等创意功能，制作人声切片等很方便。
Alter/Ego VST乐器-合成/人声 官网 ○ x 支持英语、法语和日语的歌声合成引擎，使用的是90年代较旧的技术，不过能带来一种与UTAU或Vocaloid不同的感觉。
MFreeFXBundle VST效果器 官网 ○ x 一系列常用的效果器，其中的卷积混响插件是个亮点，同类产品中很少见到免费的。
ValhallaSpaceModulator VST效果器 官网 ○ x 它可以产生镶边、延时、混响等多种效果，由于它完全没有公布原理或提供手册，不知道该怎么评价，只能说个人主观感受还不错。
ReaPlugs VST效果器 官网 ○ ? REAPER内置的一系列效果器，在其它DAW也可使用。其中的ReaJS支持脚本编程，可以自制各种想要的功能，网上也可找到其他人的作品。
Camomile VST乐器/效果器 源码 ○ ○ 本插件集成了PureData，可以直接将后者的程序作为VST运行，使人不需要接触底层代码，仅专注音频处理的原理就可以制作自己的合成器与效果器。

游戏

名称 类别 链接 免费 开源 个人评测
Godot 引擎 官网 ○ ○ 2D和3D的多平台游戏引擎，提供三种编程方式：可视化逻辑、类似Python的脚本语言以及C#。它使用基于节点的架构来实现游戏逻辑，我个人很喜欢这种思路。它与Blender的协作很方便，3D场景与动画均可一键输出到游戏引擎中。
Construct 2 引擎 官网 ? x HTML5的2D游戏引擎，不需要编程就可以制作网页应用和游戏。免费版有较严格的功能限制，基本只能用来制作小游戏。
Armory3D 引擎 官网 ○ ○ 3D多平台游戏引擎，特点是可以直接在Blender中进行所见即所得式的开发。如果美工经验胜过编程经验，或是想进行快速原型开发，这可能是一个好工具。不过它处于较早期的阶段，很少看到有人真的用它开发出完整的游戏。
