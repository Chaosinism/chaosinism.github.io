废旧项目墓场

  • 这里原本是与Bilibili视频账号Chaosinism配套发布内容的个人博客，收录了作者2018至2022上半年的若干业余项目（3D模型、开源小游戏、视频音频制作工具脚本等）。 因长时间未更新，网站的多个功能无法正常工作，文章和项目也失去实效性，甚至发现内容存在错误。因此现将网站归档，仅保留尚具参考价值的一部分内容。
  • 作者并未停止相关兴趣爱好，2022年后的新项目与重启的旧项目将迁移到另外的GitHub账号中，目前仍待规划。

Vegas MIDI脚本

[ 2018~2019 / 现由他人维护 ]

以C#编写的脚本，运行于视频编辑软件Vegas Pro中。脚本读取MIDI文件，按照音符位置插入对应的视频、音频事件，并按照音符音高对事件进行调节变换。 可用于音MAD制作与五线谱可视化视频。

另有B站作者参考本项目的部分代码，制作了用途相似但功能更为强大的脚本。若仍有该方面需求或疑问，建议联系这位作者。

源代码：GitHub
文章介绍：音MAD助手 | 五线谱可视化

SlowSampler采样器

[ 2020 / 停止更新，有可能重启 ]

基于Pure Data语言的开源VST音频采样器，在常规的变调功能外，注重音频样本的时间伸缩，并有独特的延时调整功能，将音频素材重采样至与每个MIDI音符同样的长度。

源代码：GitHub
视频介绍：BiliBili

Kinect v2 MMD补丁

[ 2019 / 停止更新 ]

针对MikuMikuDance (MMD)的动作捕捉模块仅能使用Kinect v1，而不兼容Kinect v2的问题所做的一些调研。介绍一些相关的开源项目，以及一个可能的解决方案。

组件下载：BowlRoll
文章介绍：Kinect v2与MMD动作捕捉

法线-高度贴图转换脚本

[ 2020 / 停止更新 ]

Python脚本，利用科学计算库的优化算法，将法线贴图转换为高度/深度贴图。

在此项目后，作者继续考察为2D绘画生成法线与高度3D信息的方法，推出Blender插件NijiGPen，包含一部分类似的功能。

源代码：GitHub
文章介绍：由法线贴图生成高度图的算法

MMD模型配布

[ 2018~2022 ]

《东方Project》《Love Live!》等作品的自制MMD模型。
早期模型是在たむたむす～る编辑器的基础上使用Blender修改而成；
后来改用VRoid Studio与Blender进行建模；
再之后改为全过程使用Blender进行建模。

此外也有基于真实人物的模型，它们是利用MakeHuman与Blender制作而成。

模型下载：BowlRoll
